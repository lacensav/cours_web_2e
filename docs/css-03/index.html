Rendre les images flexible

Si votre image est plus grande que la taille actuelle de la fenêtre, elle dépassera et votre page montrera des scrollbars horizontale. Pour palier à ce problème nous pouvons utiliser une propriété css que vous connaissez: max-width. Si la page est plus grande que la taille originale de l’image, l’image gardera sa taille originale. Si la page est plus petite que la taille originale, elle prendra la largeur de la page.
