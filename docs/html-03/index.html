Pourquoi ma page d’accueil doit-elle s’appeller «index.html» ?

Quand un navigateur tente d’accéder à un site via une Url, demande à accéder à un fichier se trouvant sur un serveur (celui où le site est hébergé). Mais si l’url est http://monsite.com/, cela correspond juste au dossier «pulbic_html» comme vous pouvez l’avoir sur Hostinger. Pour savoir quoi afficher, il faudrait spécifier un fichier comme http://monsite.com/a-propos.html. Si l’on ne lui dit rien, le serveur va automatiquement (par convention) chercher à l’intérieur du dossier un fichier qui s’appelle index.html.

<!-- L’adresse voulue -->

http://monsite.com/

<!-- Le fichier demandé -->

|public_html/
  |-- index.html

<!-- L’adresse voulue -->

http://monsite.com/travaux/archive.html

<!-- Le fichier demandé -->

|public_html/
  |travaux/
  |-- archive.html

<!-- L’adresse voulue -->

http://monsite.com/expositions/

<!-- Le fichier demandé -->

|public_html/
  |expositions/
  |-- index.html
